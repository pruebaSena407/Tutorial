CSS: estilos de la página

CSS permite cambiar la apariencia de los elementos HTML, como colores, tamaños y posiciones.

Ejemplo de color

<style>
p{
color:red;
}
</style>

Este texto tiene estilo aplicado con CSS.

Ejemplo de caja con CSS

<div class="caja">
Contenido de la caja
</div>
