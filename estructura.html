Estructura de un proyecto web

Un proyecto web normalmente está organizado en diferentes archivos para separar la estructura, el diseño y la lógica.

Ejemplo de estructura

tutorial-web/
│
├── index.html
├── styles.css
├── script.js
└── img/

Esta organización permite mantener el proyecto ordenado y facilitar el mantenimiento del código.
